# 📖 About

Hotel Python Palace is a hotel management system created as an educational and portfolio project.

The application demonstrates the development of a complete information system using both web and console technologies. It allows hotel staff to manage reservations, guests, rooms, invoices and occupancy while providing an intuitive web interface and a Java console administration tool.

The project was developed to strengthen practical skills in:

* Python
* Flask
* Java
* SQLite
* HTML5
* CSS3
* JavaScript
* Database Design
* Software Engineering
* Full-Stack Development

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/adrvalente/hotel-python-palace.git
```

Open the project

```bash
cd hotel-python-palace
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```text
http://127.0.0.1:5000
```
