About

Project Overview

🏨 What is Hotel Python Palace?

Hotel Python Palace is a full-stack hotel management project created to simulate the daily operations of a hotel. The system includes a Flask web application, a SQLite database and a Java console tool for administrative tasks.

The goal of this project is to demonstrate practical skills in backend development, database design, CRUD operations, authentication, responsive web design and software documentation.
